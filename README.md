# Modèle de prédiction simple : prix d'une maison
## Objectif
Créer un modèle simple pour prédire le prix d'une maison en fonction de sa surface.

## Technologies
- Python
- pandas
- numpy
- matplotlib

## Contenu du projet
- Création d'un dataset fictif (surface et prix)
- Calcul d'une régression linéaire (coefficient et intercept)
- Visualisation graphique de la régression
- Prédiction pour de nouvelles surfaces

## Résultat
- Le modèle permet de prédire le prix d'une maison selon sa surface
- Illustratif de la logique de régression linéaire utilisée en Machine Learning
